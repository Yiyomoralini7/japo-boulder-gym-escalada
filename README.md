# 🧗 Japo Boulder - Sitio Web Oficial

Sitio web responsive para el gimnasio de escalada **Japo Boulder** en Punta Arenas, Región de Magallanes.

## 📋 Descripción

Página web moderna y completamente funcional para promover el gimnasio de escalada, gestionar reservas y conectar con la comunidad.

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsive (Flexbox, Grid, Animaciones)
- **JavaScript Vanilla (ES6+)** - Interactividad y funcionalidades
- **MySQL** - Base de datos
- **PHP** - Backend para formularios

- ✅ Responsive (mobile-first)
